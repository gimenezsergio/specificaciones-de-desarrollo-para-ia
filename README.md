# ProjectSpec Pro 🚀

**ProjectSpec Pro** es una herramienta web interactiva diseñada para la creación y estructuración de especificaciones técnicas completas y "vivas", ideales para el desarrollo guiado por Inteligencia Artificial (Spec-Driven Development) usando agentes como Claude Code, Cursor, Copilot y similares.

## Características

- 📋 **Formulario Estructurado**: Guía paso a paso a través de secciones clave (Información General, Producto, Requerimientos, UX/UI, Arquitectura, Modelo de Dominio, Base de Datos, API, Testing, Roadmap y Reglas de Agentes).
- 📊 **Calidad de Especificaciones**: Panel en tiempo real que evalúa la completitud y sugiere mejoras rápidas.
- 💻 **Vista Previa de Documentos**: Previsualización instantánea de los archivos generados en formato Markdown.
- 📦 **Exportación**: Descarga en formato ZIP con la estructura de archivos sugerida o descarga de un archivo Markdown único unificado.

## Estructura de Documentos Generados

La herramienta genera y estructura los siguientes documentos clave para tu proyecto:
* `AGENTS.md` - Contexto y reglas para los agentes de desarrollo.
* `PRODUCT.md` - Visión, objetivos, métricas y alcance del producto (MVP).
* `REQUIREMENTS.md` - Matriz de permisos, requerimientos funcionales y casos de uso.
* `USER_STORIES.md` - Historias de usuario sugeridas basadas en el alcance.
* `ACCEPTANCE_CRITERIA.md` - Criterios de aceptación verificables.
* `ARCHITECTURE.md` - Patrón de diseño, stack tecnológico y estructura de carpetas.
* `DOMAIN_MODEL.md` - Entidades, relaciones y diagramas Mermaid.
* `DATABASE.md` - Motor, esquema de base de datos y sensibilidad de datos.
* `openapi.json` - Especificación de API estándar.
* `TESTING.md` - Framework de pruebas y Definition of Done (DoD).
* `ROADMAP.md` - Plan de evolución del producto.
* `DECISIONS.md` - Registro de decisiones de arquitectura (ADR).
* `RISKS.md` - Riesgos técnicos y de negocio.
* `GLOSSARY.md` - Glosario de términos del dominio.

## Cómo Usar

Simplemente abre el archivo `index.html` en cualquier navegador web moderno. No requiere de backend ni de instalaciones complejas. Toda la información se guarda localmente en el almacenamiento del navegador (`localStorage`) para que no pierdas tu progreso.
